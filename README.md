# Recipe Page

A recipe page for a simple omelette recipe. The page includes preparation time, ingredients, instructions, and nutrition information. This page has mobile and desktop versions.
HTML and CSS were used for this project as it only contains information.

## Features
- Responsive layout for mobile and desktop screens
- Preparation time section
- Ingredients list
- Step-by-step instructions
- Nutrition table
- Custom list markers and styling
- Rounded image and container on desktop

## Built With
- HTML5
- CSS3
